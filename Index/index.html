My Works

Tic-Tac-Toe Game

Tic-Tac-Toe: Championship Edition is a JavaFX-based game that I developed, incorporating advanced Object-Oriented Programming (OOP) principles. The game features an interactive UI built with Scene Builder and implements the Minimax algorithm for AI, allowing players to challenge a smart computer opponent. It supports single-player and two-player modes, ensuring an engaging experience. The project also includes a well-structured class design, using enumerations, recursion, and interfaces to enhance code efficiency and maintainability.

Pet Care Management System

This is my final project for the first semester - JavaFX-based application
designed to streamline the management of a pet care center. It allows users
to efficiently handle appointments, health records, and communication with
pet owners. The system provides features such as SMS reminders, jasper reports
for tracking of pet health information. The UI is user-friendly, built using
Scene Builder, and is designed to simplify staff operations.
