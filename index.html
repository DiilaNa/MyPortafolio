Tic-Tac-Toe Game

Tic-Tac-Toe: Championship Edition is a JavaFX-based game that I developed, incorporating advanced Object-Oriented Programming (OOP) principles. It features an interactive UI built with Scene Builder and implements the Minimax algorithm for AI. It supports single-player and two-player modes, using enumerations, recursion, and interfaces to enhance code maintainability.

Pet Care Management System

This is my final project for the first semester - a JavaFX-based application designed to streamline pet care center management. Features include appointment scheduling, health records, SMS reminders, and Jasper report generation, all managed via a Scene Builder-designed UI.
